Help Page

Welcome to the Tetrapyrrole Photochemical Spectral Database (TPSD) help page. Click each section title below to expand or collapse the content.

1. About This Database

The Tetrapyrrole Photochemical Spectral Database (TPSD) is designed to organize and provide access to chemical and spectral data for tetrapyrrole compounds. The database contains information related to compound identity, absorption spectra, fluorescence spectra, solvent conditions, fluorescence quantum yield, and literature references.

This database is intended to help users search, compare, and review spectral properties of compounds for research and educational purposes.

The home page also displays current database statistics, an update log, and external links to related literature and the Lindsey Lab.

2. What You Can Search

Users may search the database using one or more of the following criteria:

  • Compound Name – search for a specific compound by name
  • Solvent – find records measured in a specific solvent
  • Absorption Wavelength Range – search for compounds with absorption maxima in a selected range
  • Emission Wavelength Range – search for compounds with fluorescence emission maxima in a selected range

Users may enter one search condition or combine multiple filters to narrow the results.

3. How to Search the Database

3.1 Search by Compound Name

Enter the name of a compound in the Compound Name field and click the Search button. This option is useful if you already know which compound you want to examine.

3.2 Search by Solvent

Enter the solvent name to find measurements recorded in a specific solvent. This can help users compare spectral properties under different experimental conditions.

3.3 Search by Absorption Wavelength

Enter a minimum and/or maximum absorption wavelength value in nanometers (nm). The database will return compounds whose absorption maximum falls within that range.

3.4 Search by Emission Wavelength

Enter a minimum and/or maximum emission wavelength value in nanometers (nm) to identify compounds with fluorescence emission maxima in the selected range.

3.5 Combine Multiple Filters

Users may combine multiple fields in one search. Using more than one filter usually produces a smaller and more specific result set.

5. Understanding Search Results

The search results table shows compounds that match your selected filters. Each result may include the compound name, solvent, absorption maximum, emission maximum, and quantum yield.

  • Absorption λmax: wavelength where the compound absorbs light most strongly.
  • Emission λmax: wavelength where the compound emits fluorescence most strongly.
  • Quantum Yield: fluorescence efficiency of the compound, when available.

Click the View Details button to view structure images, metadata, downloadable files, and full spectrum plots.

6. Comparing Compounds

Search results include a Compare checkbox for each compound. Select two or more compounds to add them to the compare basket on the side of the search page. The basket can be expanded or collapsed while you continue searching.

  • Selections are saved while you run additional searches.
  • You can remove individual compounds from the basket.
  • You can clear all selected compounds at once.
  • Choose whether to compare absorption spectra, emission spectra, or both.

Click Compare Spectra to open the compare page. The compare page uses Chart.js to display interactive spectra and includes controls for wavelength range, normalization, line width, grid display, and spectra type.

7. Compound Detail Pages

Click View Details from a search result to open the compound detail page. The detail page shows the structure image, metadata, spectrum information, downloadable raw data files, and full spectrum plots when data files are available.

  • Missing metadata values are shown as “-”.
  • Absorption-only compounds still show absorption information and plots.
  • Full spectrum plots are generated automatically when the page loads.
  • Generated detail plots are temporary and are served from a temporary plot route.

Raw fluorescence and absorption files can be downloaded when available. These files contain wavelength and intensity or absorbance values for additional analysis.

8. Downloading Data

Raw spectrum text files are available from compound detail pages when the database has a source file for that compound and spectrum type.

  • Fluorescence data: wavelength and fluorescence intensity values.
  • Absorption data: wavelength and absorbance values.
9. Tips for Better Searching
  • Check spelling carefully when entering a compound name
  • Use wavelength ranges when exact values are unknown
  • Try fewer filters if no results are returned
  • Use the compare basket to collect compounds across several searches
  • Open compound details to confirm whether raw absorption or emission files are available
